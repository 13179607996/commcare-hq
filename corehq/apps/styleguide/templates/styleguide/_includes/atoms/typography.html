Typography

Font choice can have a strong yet subtle effect on how users perceive content.

HQ's default fonts are defined as follows: 

Avenir, Roboto, "Nunito Sans", "Helvetica Neue", Helvetica, Arial, sans-serif
Which specific font a user sees will depend on what they have installed. The first available font, reading left to right, will be used. We embed Nunito Sans, so it should be rare for someone to see anything lower in the hierarchy than that.

This set of fonts ensures consistency with:

  • Dimagi print designs, which are based on Avenir
  • Android, which uses Roboto as its default font
  • dimagi.com, which uses Nunito Sans
