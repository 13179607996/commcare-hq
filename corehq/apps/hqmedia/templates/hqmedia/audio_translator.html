This page generates files to assist in working with audio translation vendors. It generates a zip file that contains two Excel files:

The first file is for the translator's use. It contains one row for each audio file to create, with columns for the text and the desired filename. This file will contain a row for each question that has an audio filename specified but has not yet had an audio file uploaded. If multiple questions use have identical text, the file will contain only one row for that set of questions. It will arbitrarily select one of the questions' filenames.

The second file is for updating audio paths for questions with duplicate text. This file will contain a row for each question that should have its audio path changed due to the de-duplication described above. After downloading and sanity checking, re-upload this file in Bulk Application Translations.
